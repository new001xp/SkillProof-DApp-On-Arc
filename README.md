# SkillProof DApp

A decentralized certificate issuance and verification application built with Solidity, Hardhat, Ethers.js, and a simple web frontend.

## Overview

SkillProof allows organizations, trainers, educators, and individuals to issue blockchain-based skill certificates to any wallet address. Certificates are permanently stored on-chain and can be viewed by anyone.

## Features

* Connect MetaMask wallet
* Issue skill certificates
* Store certificates on-chain
* View certificates by wallet address
* Immutable blockchain records
* Solidity smart contract
* Hardhat deployment support
* Lightweight frontend

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd skillproof-dapp
```

Install dependencies:

```bash
npm install
```

## Compile Contract

```bash
npx hardhat compile
```

## Deploy Contract

```bash
npx hardhat run scripts/deploy.js
```

After deployment, copy the deployed contract address.

## Configure Frontend

Open:

```text
frontend/abi.js
```

Replace:

```javascript
const CONTRACT_ADDRESS = "PASTE_DEPLOYED_CONTRACT";
```

with your deployed contract address.

Also replace the ABI with the compiled contract ABI generated by Hardhat.

## Run Frontend

Open:

```text
frontend/index.html
```

in a browser with MetaMask installed.

## Smart Contract Functions

### issueCertificate

Issues a certificate to a wallet.

Parameters:

* user (address)
* skill (string)
* issuer (string)

### getCertificates

Returns all certificates belonging to a wallet.

Parameters:

* user (address)

Returns:

```solidity
Certificate[]
```

## Certificate Structure

```solidity
struct Certificate {
    string skill;
    string issuer;
    uint256 issuedAt;
}
```
